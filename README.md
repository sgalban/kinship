# Description

A small project made in collaboration with graphic designer [David Comberg](https://davidcomberg.com/) for the Wolf Humanities Center [2019-2020 Forum on Kinship](https://wolfhumanities.upenn.edu/annual-topics/kinship).
This respository consists of several patterns generated with [Processing](https://processing.org/) 3.3.6, one of which was selected by David, iterated upon, and used in his designs for the forum's printed materials.
